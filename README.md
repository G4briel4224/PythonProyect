# 📰 TuPrimeraPaginaApellido - Proyecto Final Django

Este proyecto es una aplicación web estilo blog construida con Django. Permite a los usuarios crear publicaciones, ver el perfil de otros usuarios, enviar mensajes, y mucho más.

---

## 🚀 Funcionalidades principales

- ✅ Registro, login y logout de usuarios
- ✅ Creación, edición y eliminación de publicaciones (tipo blog)
- ✅ Carga de imágenes e integración con CKEditor
- ✅ Vista detallada de cada publicación
- ✅ Búsqueda de publicaciones por título
- ✅ Categorías para organizar las publicaciones
- ✅ Perfil de usuario con avatar, biografía, nombre, apellido y cambio de contraseña
- ✅ Página "Acerca de mí"
- ✅ Sistema de mensajería entre usuarios
- ✅ Navbar con acceso a todas las vistas principales
- ✅ Uso de CBV (Clases Basadas en Vistas), Mixins y decoradores
- ✅ Herencia de templates

---

## 🧑‍💻 Tecnologías utilizadas

- Python 3
- Django 5.2.1
- Bootstrap 5
- CKEditor
- SQLite (en desarrollo local)
- HTML/CSS
- Git & GitHub
